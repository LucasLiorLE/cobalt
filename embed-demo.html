Generated Open Graph Meta Tags

og:description This is a sample YouTube video processed by Cobalt for Discord embed testing.
og:type video.other
og:video:url https://example.com/video/youtube/dQw4w9WgXcQ.mp4
og:video:type video/mp4
og:video:width 1280
og:video:height 720
og:image https://img.youtube.com/vi/dQw4w9WgXcQ/maxresdefault.jpg
og:site_name Cobalt
video:duration 180

How Discord Will Display This

When you share this URL in Discord:

  1. Discord fetches the Open Graph meta tags
  2. Shows the video title and description
  3. Displays the thumbnail image
  4. Provides a link to the video file
  5. Users can click to play the video directly in Discord

Example URLs:

  • /embed/youtube/dQw4w9WgXcQ
  • /embed/twitter/1234567890
  • /embed?url=https://youtube.com/watch?v=dQw4w9WgXcQ
